# 세법
※ 답안 작성시 유의사항
1. 답안은 문제 순서대로 작성할 것
2. 계산문제는 계산근거를 반드시 제시할 것
3. 답안은 아라비아 숫자로 원단위까지 작성할 것
   (예 : 2,000,000 - 1,000,000 = 1,000,000원)
4. 별도의 언급이 없는 한 관련 자료․증빙의 제출 및 신고․납부절차는 적법하게 이행된 것으로 가정할 것

## 【문제 1】(24점)
