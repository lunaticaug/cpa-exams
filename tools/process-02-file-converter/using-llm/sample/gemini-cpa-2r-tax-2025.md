# 세법

※ 답안 작성 시 유의사항
1. 답안은 문제 순서대로 작성하시오.
2. 계산문제는 계산근거를 반드시 제시하시오.
3. 답안은 아라비아 숫자로 원단위까지 작성하되, 원 이하는 절사하시오(예: 1,000,000.6원 → 1,000,000원).
4. 별도의 언급이 없는 한 관련 자료·증빙의 제출 및 신고·납부절차는 적법하게 이행된 것으로 가정하시오.
5. 별도의 언급이 없는 한 합법적으로 세금부담을 최소화하는 방법으로 풀이하시오.
