| 구분 | 주식 X | 주식 Y | $F_1$ | $F_2$ |
|------|--------|--------|-------|-------|
| 평균수익률 | 20% | 10% | 15% | 5% |
| 표준편차 | 20% | 15% | 10% | 20% |

| 단일모형 $β_{i1}$ | ② | 0.7 | - | - |
| 단일모형 $R^2$ | 0.64 | ④ | - | - |

| 2요인모형 $β_{i1}$ | 1.5 | 0.9 | - | - |
| 2요인모형 $β_{i2}$ | (-)0.4 | (-)0.5 | - | - |
